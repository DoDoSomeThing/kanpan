## 背景:為什麼是這 4 個

2026-06-21 評估:vp_score / 判讀燈號**已驗證無勝率 edge**(四檔實測 20日桶勝率全 47-48% <50%,V2_spec 亦載 90-100 桶僅 45%)。
→ 結論:**別再加預測力**(自我欺騙)。該加的是「**防止跑輸大盤 + 防止行為犯錯 + 誠實衡量**」。

## ~~P2 — 組合層曝險~~ ✅ 已完成 2026-06-21

**問題**:面板現在是**單股**。但常看的四檔(2356/3231/6669/2382)全 AI server 代工,**高度相關 = 假分散**。

**已做**(新 `portfolio.py` + api `/portfolio` + CLI `python portfolio.py [--cash 元]`):
- 持倉間**日報酬相關係數矩陣**(揭露假分散);相關 >=0.7 列「高相關對」警告。
- **曝險權重**:各檔市值佔比 + 總市值;`--cash`/`?cash=` 選填 → 投資比%。
- test_portfolio 釘 24 案(pearson/對齊/矩陣/曝險)。

**砍**:整體產業/個股集中度 → **無產業分類資料**(names.json 只有股名),硬做需外部來源,先略。

## ~~P3 — 行為守門~~ ✅ 已完成 2026-06-21

**問題**:對應用戶弱點(有持倉、手癢想短線)。

**已做**(新 `behavior.py`,只警示不下指令;面板/擴充顯示):
- **追高警示**:無持倉 + 月線乖離 >=15% 或 60日位置 >=85% → 「期望值差,離套牢區近」。
- **凹單偵測**:已跌破生效出場(🔴 已觸發)仍持有 → 提醒(trail 是唯一真 edge,別凹)。
- **頻率警示**:近 30 日平倉 >4 次 → 「過度交易,摩擦成本吃報酬」。
- test_behavior 釘 13 案(各警示觸發/不誤報/彙整)。

## ~~P4 — 勝率桶樣本數 + 信賴區間~~ ✅ 已完成 2026-06-21

**問題**:47% 勝率 —— **幾筆算的?** 30 筆的 47% 跟 3000 筆的 47% 天差地別。

**已做**:

- test_core 加 7 案(對稱/小樣本更寬/邊界鉗/None 防呆)。
