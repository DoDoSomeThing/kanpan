# Fib 速查筆記 — 怎麼抓波段高低

## 核心 5 點

1. **Fib 只量「一段」**：要嘛一段漲、要嘛一段跌，不是量整張圖。中間不能掉頭好幾次（掉頭 = 跨太多段，重抓）。
2. **漲段低在左、跌段高在左**：
   - 漲段：高 = 近期頂（右）、低 = 起漲底（左）→ Fib 找「回檔買點」
   - 跌段：高 = 起跌頂（左）、低 = 近期底（右）→ Fib 找「反彈壓力 / 別接刀」
3. **高低點 = 左右最突出的尖頂尖底**：肉眼一看就跳出來，要瞇眼找 = 雜訊，別用。
4. **錨點抓越近 → 支撐壓力越貼現價**：大段給格局（支撐可能 -12%），近段給短線進出（支撐 -6%）。
5. **工具告訴你「卡在哪」，不告訴你「會不會動」**：最大用處是攔你別追高 / 別接刀。

## 抓「真頂 / 真底」的眼力

- **真底** = 左右兩根都比它高（漲股的起漲底、跌股的近期底）
- **真頂** = 左右兩根都比它矮（漲股的近期頂、跌股的起跌頂）
- 別抓「反彈/回升路上的某一根」當頂底 —— 那不是轉折。
  - 例：3481 6/11 低 40.25（真底）；6/12 低 48.5 是反彈第二根，不是底。

## 影線 vs 收盤

同一根插長影時兩種抓法都有人用：
- 抓「插針最深 / 最高」→ 用影線高低（如 40.25）
- 抓「收盤站穩」→ 用收盤價（如 46.95）
極端撐壓用影線，穩健用收盤。知道差別即可。

## 換錨點結論還一樣 → 訊號最可信

3481 不管填大段(22.95)或近段(40.25)，「位置」都是「高檔強勢區」= 兩種抓法都同意貼頂。
這種**換錨點結論不變**的時候，判讀最可信。

## 跌股獨有陷阱

跌股常還在創新低（沒止跌）。此時「低」只是「今天最低」，明天可能更低、要重填。
- 跌股 Fib **只信「反彈壓力」**（彈到哪卡住）
- **不信「底在哪」**——底要等「不再破低 + 站回 MA」才算
- 面板「瀕臨破底 / 已破底」= 底還沒到

## 一句話總結

漲股的 Fib 找回檔買點；跌股的 Fib 找反彈賣點 / 別接刀。**方向相反，左右對調。**
