# kanpan 看盤

台股看盤面板。**不預測漲跌**，只回答五件事：

> 現在趨勢如何？什麼結構？動能強不強？量能如何？**歷史上類似狀況表現如何？**

不輸出「買進/賣出」，輸出「風險評估 / 趨勢評估 / 歷史勝率參考」。
判斷留給人，工具負責把盤面攤開。

## 為什麼這樣設計

前身專案用嚴謹 walk-forward 驗證了 6 種選股訊號（技術評分／營收驚奇／法人連買／
均值回歸／動能／相對強度），**全數無 alpha**——簡單公開訊號選股打不贏指數。
所以 kanpan 不做第 7 次嘗試，改做唯一驗證過有價值的事：**看懂現況 + 誠實的歷史統計**。

## 結構

```
core.py                  指標 + 四區塊 + VP Score（純函式，無 I/O 副作用）
panel.py                 CLI 面板：python panel.py 2330
research/score_history.py  歷史驗證：對 2021-2024 全市場 37.5 萬樣本分桶統計
data/fetch_data.py       下載資料（大檔不進 git，放 GitHub Release）
```

## 快速開始

```bash
python data/fetch_data.py cache   # 近期日K（面板用）
python panel.py 2330              # 看面板
# （可選）重建歷史統計：
python data/fetch_data.py deep    # 80MB 深歷史K
python research/score_history.py
```

## 誠實的發現（2021-06 ~ 2024-12，375k 樣本）

| Score | 樣本 | 20日勝率 | 平均20日 |
|---|---|---|---|
| 90-100 | 5,321 | 46.1% | **+1.69%** |
| 80-89 | 29,334 | 47.8% | +1.66% |
| 50-59 | 63,651 | 48.3% | +0.83% |
| 0-39 | 103,167 | 49.4% | +0.66% |

高分桶**勝率沒有比較高**（甚至略低），但**平均報酬較高**＝右偏分布：
強勢結構輸的次數不少，但贏的時候贏比較多（動能特性）。
這不是聖杯，是描述。最大回撤欄提醒你：任何分數都可能 -40%~-80%，部位與停損自理。

## 不做的事

AI 預測漲跌、買賣訊號、營收驚奇、新聞情緒、法人模型——
要嘛已驗證無 alpha，要嘛工程量大而第一版價值低。
